# About this portfolio

This site collects hands-on digital-forensics projects across the discipline —
**data recovery**, **mobile** and **host artifact analysis**, **timeline
reconstruction**, and the **custom tooling** that supports an examination. Each
project is a short writeup backed by a self-contained, runnable tool. The set
grows over time and isn't limited to any single subdomain.

### How the work is done

- **Real, reproducible output.** Tools are run against legitimate sample data and
  the genuine results are captured and shown. Nothing is fabricated.
- **Auditable code.** Tool code is pure Python standard library wherever
  possible, so it runs with no third-party install and can be read end to end.
- **Documented provenance.** Every dataset is either self-generated synthetic
  data or a public practice dataset, and the source is stated in the writeup.

### Integrity

No real casework appears here — nothing from prior employers, real cases, or
actual subjects. All data is synthetic or drawn from public practice corpora.
Project history is honest and current-dated.
